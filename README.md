# imview
A simple image viewer for the urxvt terminal, written in zsh.

![Screenshot](https://github.com/GasparVardanyan/imview/raw/master/screenshot.png)

## Keybindings
* **+**,**-**,**=** - *zoom in, out, reset*
* **h**,**j**,**k**,**l** - *vi-like movement*
* **c** - *move to centre*
* **0** - *reset zoom and position*
* **q** - *quit*
